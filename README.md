# Osu! Music Player

A simple WPF desktop app to browse and play your local osu! song library.

<img width="800" height="450" alt="osu_music_wpf_W9iURFbxtJ" src="https://github.com/user-attachments/assets/4827d60c-cecd-4bc9-88a9-55ab46268712" />

## Features

- Browse and play songs from your Osu! library
- Play, pause, and skip tracks
- DT (Double Time) and NC (Night Core) speed modifiers

## Requirements

- Windows 10 or later
- .NET 10
- Osu! installed locally

## Usage

1. Clone the repo and open `osu music wpf.slnx` in Visual Studio
2. Build and run
3. The app will automatically detect your osu! Songs folder
